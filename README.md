If you want to run this project on your local machine, just follow these simple steps:

1. Clone the repository
   git clone https://github.com/Rohithraj16/tiktok-player-Rohithrai016.git
   cd tiktok-player-Rohithrai016
2. Install dependencies

Make sure you have Node.js installed (version 18 or higher is recommended)

npm install 3. Start the development server
npm run dev 4. Open the app

Once the server is running, open your browser and go to:

http://localhost:5173
